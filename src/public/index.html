Funciones de la API

Listar todos los usuarios

Método: GET

Endpoint: http://3.239.163.221:3000/users

Buscar usuarios por ID

Método: GET

Endpoint: http://3.239.163.221:3000/users/:id

Crear un nuevo usuario

Método: POST

Endpoint: http://3.239.163.221:3000/users

Body:

{
  "name": "Juan Gonzalez",
  "email": "juan.gonzalez@gmail.com"
}

Borrar un usuario

Método: DELETE

Endpoint: http://3.239.163.221:3000/users/:id

Actualizar un usuario

Método: PUT

Endpoint: http://3.239.163.221:3000/users/:id

Body:

{
  "name": "Juan Gonzalez",
  "email": "juan.gonzalez@gmail.com"
}
